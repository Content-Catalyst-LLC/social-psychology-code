# Measurement Notes

## Helping propensity

A simplified helping-propensity model:

H* = beta0
   + beta1*Empathy
   + beta2*PerspectiveTaking
   + beta3*NormSalience
   + beta4*ReciprocityExpectation
   + beta5*PerceivedEfficacy
   + beta6*FeltResponsibility
   + beta7*IdentityOverlap
   + beta8*Trust
   + beta9*MoralIdentity
   - beta10*HelpingCost
   - beta11*InterventionRisk
   - beta12*log(BystanderCount + 1)

## Prosocial behavior versus altruism

Prosocial behavior is the broader behavioral class.
Altruism is a narrower motivational subtype involving costly other-regarding concern.

## Bystander suppression

Bystander count can suppress helping by reducing felt responsibility:

Responsibility = f(-log(bystanders + 1), direct assignment, urgency, identity, norm salience)

## Public-goods contribution

Contribution = f(norm salience, efficacy, trust, institutional legitimacy, moral identity, reciprocity expectation, cost)

## Organizational citizenship behavior

Organizational prosociality should distinguish:

- helping individuals
- civic virtue
- courtesy
- conscientiousness
- voice
- mentoring
- knowledge sharing

## Recommended reliability checks

- Empathy scale reliability
- Perspective-taking reliability
- Norm salience reliability
- Perceived efficacy reliability
- Felt responsibility reliability
- Institutional legitimacy reliability
- Moral identity reliability

## Recommended validity checks

- Verify helping-cost manipulation.
- Verify norm-salience manipulation.
- Verify efficacy manipulation.
- Include actual behavioral outcomes when possible.
- Test whether empathy predicts helping after controlling for reputation and reciprocity.
- Test whether institutional legitimacy moderates cooperation.
