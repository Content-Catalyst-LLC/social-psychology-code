# Measurement Notes

## Basic attributional asymmetry

FAE_i = D_i - S_i

where D_i is dispositional attribution and S_i is situational attribution.

## Constraint neglect score

CN_i = actual_constraint_i - perceived_constraint_i

Higher values indicate under-recognition of situational constraint.

## Correspondence bias score

CB_i = correspondence_inference_i - choice_freedom_i

When observers infer strong correspondence despite low choice freedom, correspondence bias is high.

## Accountability correction

FAE_i = beta0 + beta1 * cognitive_load_i - beta2 * accountability_i
        - beta3 * perspective_taking_i - beta4 * constraint_salience_i + error_i

## Blame pathway

Punishment_i = beta0 + beta1 * dispositional_attribution_i
               - beta2 * situational_attribution_i
               + beta3 * moral_blame_i + error_i

## Recommended reliability checks

- Attribution scale reliability
- Constraint perception scale reliability
- Moral blame and punishment scale reliability
- Manipulation checks for choice freedom and cognitive load
- Response-time filtering rules

## Recommended validity checks

- Compare constrained and unconstrained behavior
- Compare observer judgments with actor self-reports
- Test whether constraint salience reduces dispositional attribution
- Test whether cognitive load increases dispositional inference
- Test whether accountability reduces correspondence bias
- Test cultural orientation and role-based reasoning as moderators
