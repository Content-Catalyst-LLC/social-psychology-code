# Measurement Notes

## Utility model of altruism

U_i = x_i - c + alpha*x_j

where:

- x_i = actor's own payoff
- c = cost of helping
- x_j = recipient welfare
- alpha = weight placed on the other's welfare

Helping is predicted when:

alpha * Delta(x_j) > c

## Empathy-altruism model

Helping propensity can be represented as:

H* = beta1*EmpathicConcern
   + beta2*RecipientNeed
   + beta3*PerceivedEfficacy
   + beta4*MoralIdentity
   + beta5*SocialNormSalience
   - beta6*HelpingCost
   - beta7*PersonalDistress
   + beta8*ReputationVisibility
   + beta9*WarmGlowExpectation

The empathy-altruism claim implies that empathic concern remains predictive even when escape is easy and reputation visibility is low.

## Kin selection

Hamilton's rule:

rB > C

where:

- r = relatedness
- B = benefit to recipient
- C = cost to actor

## Reciprocal altruism

A simplified repeated-interaction condition:

p_future * B_future - C > 0

where p_future is the perceived probability of future reciprocity.

## Public goods and altruistic punishment

Contribution_i = f(norm_salience, efficacy, trust, moral_identity, punishment_risk, free_rider_expectation)

Punishment_i = f(norm_violation, anger, fairness_norm, punishment_cost, group_benefit)

## Recommended reliability checks

- Empathic concern scale reliability
- Personal distress scale reliability
- Moral identity scale reliability
- Social norm salience reliability
- Perceived efficacy reliability
- Warm-glow expectation reliability
- Reciprocity expectation reliability

## Recommended validity checks

- Verify empathy manipulation increased empathic concern rather than only distress.
- Verify easy-escape manipulation reduced egoistic escape pressure.
- Verify anonymity manipulation reduced reputation visibility.
- Verify cost manipulation changed perceived cost.
- Test whether empathy predicts helping when reputation and reciprocity are controlled.
- Test whether public-goods contributions respond to efficacy and norm salience.
