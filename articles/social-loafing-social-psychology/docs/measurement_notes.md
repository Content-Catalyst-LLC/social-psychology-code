# Measurement Notes

## Basic group production

Q_g = sum(e_i) - C_g

where Q_g is group output, e_i is member effort, and C_g is coordination cost.

## Loafing intensity

L_i = e_i_solo - e_i_group

Positive values indicate lower effort in the collective condition.

## Collective Effort Model

Effort is expected to increase when people believe:

1. individual effort improves individual performance,
2. individual performance improves group performance,
3. group performance leads to valued outcomes.

A simplified model:

Effort_i = β0
  + β1 Identifiability_i
  + β2 Accountability_i
  + β3 TaskValue_i
  + β4 Instrumentality_i
  + β5 TaskUniqueness_i
  - β6 log(GroupSize_i)
  - β7 FreeRiderExpectation_i
  - β8 SuckerEffectConcern_i
  + β9 SocialCompensation_i

## Process-loss decomposition

PotentialOutput - ActualOutput = CoordinationLoss + MotivationLoss

Social loafing concerns the motivation-loss component.
