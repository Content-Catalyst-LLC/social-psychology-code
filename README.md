# Social Psychology Code

Open computational scaffolding for the Social Psychology knowledge series.

This repository supports article-level examples for social cognition, schemas, attribution, impression formation, attitudes, persuasion, conformity, obedience, social norms, social identity, intergroup relations, prejudice, prosocial behavior, cooperation, aggression, relationships, leadership, organizations, misinformation, social media, institutional trust, and collective action.

## Structure

Each article folder contains:

- `python/`
- `r/`
- `julia/`
- `cpp/`
- `fortran/`
- `c/`
- `rust/`
- `go/`
- `sql/`
- `notebooks/`
- `docs/`
- `data/raw/`
- `data/processed/`
- `outputs/`
